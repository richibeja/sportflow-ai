# SportFlow AI | Extreme Ultra v5.0 ⚡⚽

![SportFlow AI](sportflow_hero_bg.png)

**SportFlow AI** is a premium sports streaming and analysis platform designed for maximum user engagement and monetization. Featuring the "SPORT-GURU"—a context-aware artificial intelligence that provides real-time betting insights and match predictions.

## 🚀 Key Features

- **Double-Unlock Stream Engine**: A high-conversion monetization flow (Step 1 -> Step 2 -> Signal) optimized for Adsterra and other ad networks.
- **SPORT-GURU (The Brujo)**:
  - **Match Awareness**: Dynamically analyzes the active match (Home vs Away).
  - **Live Intel**: Simulated deep-search (news & stats) for unmatched realism.
  - **Score-Aware Prediction**: Respects live partial scores to ensure baseline mathematical accuracy.
  - **Status Logic**: Professional checks to prevent "predicting" finished matches.
- **Viral Growth Engine**: Built-in "Share" functionality using the Web Share API with social media fallback.
- **Vercel Optimized**: Pre-configured `vercel.json` for extreme performance and clean URLs.
- **Responsive Design**: Elegant, glassmorphic UI that looks stunning on mobile and desktop.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (ES6+).
- **APIs**: ESPN Scoreboard API for real-time match data.
- **Analytics**: "Genial" internal logic for simulated AI analysis.
- **Hosting**: Optimized for **Vercel**.

## 💰 Monetization

The project is pre-configured with:
- **SmartLink Integration**: Direct access to high-CPM direct links.
- **Responsive Ad Slots**: Dedicated spaces for Desktop and Mobile banners.
- **Ad-Gated Features**: Monetized access to both the stream and the premium Guru analysis.
